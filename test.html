Komputer Avadanliqlari Haqqinda

Əsas hissə, əlavə qurğular, şəbəkə və timeline ardıcıllığı.

Esas Avadanliqlar

Prosessor (CPU)Bütün əməliyyatların mərkəzi idarəetmə nöqtəsidir.
Operativ Yaddas (RAM)Aktiv proseslərin sürətli işləməsi üçün müvəqqəti yaddaş.
Saxlama (SSD/HDD)Faylların daimi saxlanması və sistemin açılış sürəti.
Qrafik Karti (GPU)Vizual render və ağır qrafik işlərdə performans elementi.

Elave Qurgular

MonitorVizual çıxış və rəng dəqiqliyi üçün əsas ekran cihazı.
KlaviaturaMətn və komanda girişini təmin edən input qurğusu.
SicanNaviqasiya və idarə üçün dəqiq hərəkət cihazı.
PrinterRəqəmsal məlumatların fiziki nüsxəyə çevrilməsi.

Sebeke Infrastrukturu

EthernetKabelli bağlantı standartıdır; stabil və aşağı gecikməli data ötürür.
RouterŞəbəkələr arasında trafik yönləndirir və internet çıxışını paylayır.
SwitchLokal şəbəkədə cihazları birləşdirir və paketləri doğru porta ötürür.
ServerFayl, tətbiq və verilənlər bazası xidmətlərini mərkəzləşdirilmiş şəkildə təqdim edir.
ModemISP xəttindən gələn siqnalı şəbəkədə istifadə üçün uyğunlaşdırır.
FirewallTrafiki filtr edərək icazəsiz giriş cəhdlərinə qarşı qoruma yaradır.

Haqqinda

Daxili avadanlıqlar kompüterin hesablamasını, sürətini və dayanıqlığını müəyyən edir.
Xarici avadanlıqlar istifadəçi təcrübəsini və gündəlik idarəni formalaşdırır.
Şəbəkə avadanlıqları (ethernet, router, switch, server) əlaqəni, təhlükəsizliyi və xidmətlərin davamlılığını təmin edir.
